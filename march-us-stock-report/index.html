市场分析

🟢 利好因素

1. AI浪潮持续推动科技股上涨，英伟达、Meta等龙头表现强劲
2. 美联储维持利率不变，市场预期年内仍有降息空间
3. 美国经济基本面稳健，GDP增速超预期
4. 企业盈利表现良好，Q1财报季预期乐观

🔴 风险因素

1. 通胀数据反复，核心CPI仍高于目标
2. 中美贸易摩擦持续，影响中概股表现
3. 地缘政治风险犹存，能源价格波动
4. 小盘股表现疲软，市场广度不足
