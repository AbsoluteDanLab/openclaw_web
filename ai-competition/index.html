集团AI应用创意大赛

AI always stands with Human.

本方案旨在通过组织集团级AI应用创意大赛，激发各业务单元员工主动探索AI在实际工作中的应用，推动AI从认知走向实践。活动以真实业务场景为切入点，鼓励员工围绕日常工作中的效率痛点，利用AI工具进行优化与创新。

立即参与 了解工具
一

活动背景

随着AI技术的发展，越来越多的AI工具已经可以在日常工作中发挥作用

自动生成报告

AI可以快速生成各类业务报告，节省大量撰写时间

数据分析辅助

利用AI进行数据洞察和分析，发现业务机会

文案与内容生成

快速生成营销文案、邮件、公告等各类内容

会议纪要整理

自动整理会议内容，提取关键信息和待办事项

但在当前阶段，集团内部仍有大量同事处于观望状态——不清楚AI能做什么，不知道如何结合自己的工作使用。为此，集团发起本次AI应用创意大赛。

二

活动目标

本次活动的核心目标不是"比赛本身"，而是推动实际应用

激发AI使用氛围

让更多同事开始主动尝试AI工具，形成全员参与的氛围

连接真实工作场景

从日常工作中发现可以用AI优化的环节，解决实际问题

帮助员工提升效率

减少重复劳动，提高工作质量与效率，释放创造力

三

参与方式

参与对象

  • 集团各BU及职能部门员工均可参与
  • 支持个人报名
  • 支持团队报名（2-5人）
  • 建议围绕同一个业务场景组队参与

一句话参与方式

找到你工作中一个重复、耗时、低效的环节，用AI把它优化掉。

只要做到这一点，就可以作为参赛作品。

四

可以做什么

本次活动不限定形式，只要对工作有帮助，都可以参与

一个AI小工具

构建一个能解决具体问题的AI工具

一个AI助手（Agent）

创建智能助手帮助完成特定任务

一个AI自动化流程

设计自动化工作流提升效率

AI方法（Prompt/SKILL）

总结一套好用的提示词或技能

示例（用于启发）

用AI自动生成周报
用AI做门店经营分析
用AI生成营销文案
用AI整理会议纪要
用AI分析Excel数据
用AI做选址辅助分析
门店Q&A智能问答
数据分析提示词集

不要求复杂，关键是"能用、好用、有帮助"。

五

推荐工具

本次活动不限制工具平台，可自由选择

AI应用开发工具

  • Coze — Agent / 工作流 / 自然语言编程 / 低代码应用
  • WorkBuddy / OpenClaw — 龙虾产品，可处理本地文件、多Agent协作
  • 飞书AI — 多维表、机器人、Aily、秒搭
  • 企业微信 — 智能表、机器人

AI对话工具

  • 千问 — 阿里云大模型
  • Kimi — 长文本处理专家
  • 豆包 — 字节跳动AI助手
  • ChatGPT — OpenAI对话模型

其他工具

  • Python — 自动化脚本
  • RPA工具 — 流程自动化
  • Excel + AI — 数据处理增强

原则：优先选择简单、容易上手、大部分人可接触的工具

六

作品提交要求

已简化、强引导，不需要复杂材料，清楚说明即可

1

你解决了什么问题

请说明：

  • 你的工作场景是什么
  • 原来哪里低效 / 花时间
2

你是怎么用AI解决的

请说明：

  • 用了什么AI工具
  • 大致是怎么做的
3

展示你的成果

可以用：

  • 截图 / Demo / 简单视频 / 使用前后对比
4

效果如何

简单说明：

  • 是否更快了 / 更轻松了 / 更规范了
七

八

评审机制

初评阶段

  • 数科团队
  • 各BU相关负责人
  • 筛选出优秀作品

终评阶段

  • 集团管理层
  • 业务负责人
  • 董事长参与
  • 评选最终优秀案例

评审标准

  • 真实场景 — 是否来自真实工作场景，是不是日常工作中的真实问题
  • 实际帮助 — 是否确实提升效率或减少工作量
  • 简单可用 — 是否容易理解、容易上手，而不是复杂但难用
  • 创意思考 — 是否体现出对AI的合理使用和思考
  • 可推广性 — 是否可以在类似场景推广，让更多人使用

十

思考引导

我能不能报名？

如果你不知道做什么，想一想你工作中：

• 最重复的一件事

• 最花时间的一件事

• 最容易出错的一件事

然后问自己：

能不能用AI把它变简单一点？

这，就是一个参赛作品。
